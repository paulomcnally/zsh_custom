# The vdocker Plugin

Allow to run Docker on Virtual Machine.

```
$ vdocker start
``

### Clone
```
$ cd ~
$ git clone https://github.com/paulomcnally/zsh_custom.git
```

### Setup

Edit `.zshrc` file.

```
$ vi ~/.zshrc
```

Find `ZSH_CUSTOM` and set.

```
ZSH_CUSTOM=~/zsh_custom
```

Find `plugins=()` and add `vdocker`.

```
plugins=(
  vdocker git
)
```
