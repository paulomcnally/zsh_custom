# The ciclo Plugin

Plugin to work with Ciclo internal project.

```
$ ciclo start
``

### Clone
```
$ cd ~
$ git clone https://github.com/paulomcnally/zsh_custom.git
```

### Setup

Edit `.zshrc` file.

```
$ vi ~/.zshrc
```

Find `ZSH_CUSTOM` and set.

```
ZSH_CUSTOM=~/zsh_custom
```

Find `plugins=()` and add `ciclo`.

```
plugins=(
  ciclo git
)
```
